# M003 — Ramsey 数 R(5,5):下界を押し上げる

## 問題

R(5,5) は、完全グラフ K_n の辺のどんな 2-彩色も単色の K_5 を含むような
最小の n です。100 年を超える Ramsey 理論の歴史の中で、分かっているのは
**43 ≤ R(5,5) ≤ 46** だけです(下界は Exoo 1989、上界は Angeltveit & McKay
2024、[arXiv:2409.15709](https://arxiv.org/abs/2409.15709))。

単色 K_5 を含まない K_n の辺 2-彩色は R(5,5) > n の証明になります。
43 頂点以上の有効な彩色は、見出しを飾る数学的成果になるでしょう。

## スコア

`score = n`:頂点数。n 頂点上の有効な彩色は R(5,5) ≥ n + 1 を証明します。

## Witness フォーマット

```json
{
  "mission": "M003-ramsey-5-5",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 30,
  "witness": {
    "rows": ["01101...", "1011...", ...]
  }
}
```

- `rows[i]` は長さ n − 1 − i の文字列:辺 (i, i+1), (i, i+2), …, (i, n−1)
  の色——隣接行列の上三角、頂点は 0 始まり、`'0'` = 赤、`'1'` = 青。
- `rows` は n − 1 個の要素を持ちます(頂点 n−1 に行はありません)。
- どの 5 頂点についても、その間の 10 辺がすべて同色になってはいけません。

検証:`python3 verify.py <witness.json>`

## 文献記録

R(5,5) ≥ 43:Exoo の 42 頂点彩色(1989)。再構成については *Study of Exoo's
Lower Bound for Ramsey number R(5,5)*
([arXiv:2212.12630](https://arxiv.org/abs/2212.12630))を参照。この witness は
まだ `records/` にありません——有効な 42 頂点彩色を提出すれば検証済み記録に
なります。

## 既知の手法

- 辺彩色上の局所探索(単色 K_5 の個数をエネルギーとする焼きなまし/タブー
  サーチ)で 30 台半ばに到達します。
- 最良の構成は代数的です:Paley 型グラフやブロック巡回彩色に局所修復を
  組み合わせます。Exoo の 42 頂点 witness も巡回構成に基づいています。
- 高速な着手評価が重要です:共通近傍ビットマスクの技法で、各辺を通る単色 K_5
  の数を管理しましょう。
