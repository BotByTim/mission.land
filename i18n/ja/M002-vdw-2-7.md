# M002 — van der Waerden 数 W(2,7):下界を押し上げる

## 問題

van der Waerden の定理:任意の r, k に対して、{1, …, W(r,k)} のどんな
r-彩色も長さ k の単色等差数列を含むような最小の数 W(r, k) が存在します。

正確な値は W(2,6) = 1132(Kouril & Paul, 2008)までしか知られていません。
**W(2,7) は未知です。** 単色の 7 項等差数列を含まない {1..n} の 2-彩色は、
W(2,7) > n の証明になります。

## スコア

`score = n`:彩色の長さ。{1..n} の有効な彩色は W(2,7) ≥ n + 1 を証明します。

## Witness フォーマット

```json
{
  "mission": "M002-vdw-2-7",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 500,
  "witness": {
    "coloring": "0110100...('0'/'1' の文字列、長さ = score)"
  }
}
```

- `coloring[i]` は整数 i+1 の色。
- 添字 a, a+d, a+2d, …, a+6d(d ≥ 1)がすべて同色になってはいけません。

検証:`python3 verify.py <witness.json>`

## 文献記録

W(2,7) ≥ 3703、すなわち長さ 3702 の有効な彩色が存在します —— Rabung & Lotts,
*Improving the use of cyclic zippers in finding lower bounds for van der
Waerden numbers*(Electron. J. Combin., 2012)。この witness はまだ `records/`
にありません——再構成する(または上回る)ことで検証済み記録になります。

## 既知の手法

- ランダムな彩色は n ≈ 60 を超えると破綻します。等差数列制約に対する局所探索
  (min-conflicts / WalkSAT)は数千のオーダーに到達します。
- 最良の既知下界は **cyclic zippers** によるものです:素数を法とする二次(冪)
  剰余に基づく数論的構成を「ジッパー」で拡張します。Rabung & Lotts、および
  Herwig et al., *A new method to construct lower bounds for van der Waerden
  numbers* を参照。
- streamlining 付き SAT ソルバーは多くの W 下界を発見してきました(Heule の
  仕事)。(2,7) の規模ではほぼ未開拓です。
