# M003 — Ramsey 수 R(5,5): 하한 끌어올리기

## 문제

R(5,5)는 완전 그래프 K_n의 어떤 간선 2-채색도 단색 K_5를 포함하게 되는 가장
작은 n입니다. 100년이 넘는 Ramsey 이론의 역사에서 우리가 아는 것은
**43 ≤ R(5,5) ≤ 46**이 전부입니다(하한 Exoo 1989, 상한 Angeltveit & McKay
2024, [arXiv:2409.15709](https://arxiv.org/abs/2409.15709)).

단색 K_5가 없는 K_n의 간선 2-채색은 R(5,5) > n의 증명입니다. 43개 이상의
정점에서 유효한 채색을 찾는다면 헤드라인급 수학적 성과가 될 것입니다.

## 점수

`score = n`: 정점 수. n개 정점 위의 유효한 채색은 R(5,5) ≥ n + 1을 증명합니다.

## Witness 형식

```json
{
  "mission": "M003-ramsey-5-5",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 30,
  "witness": {
    "rows": ["01101...", "1011...", ...]
  }
}
```

- `rows[i]`는 길이 n − 1 − i의 문자열: 간선 (i, i+1), (i, i+2), …, (i, n−1)의
  색 — 인접 행렬의 상삼각, 정점은 0부터 번호, `'0'` = 빨강, `'1'` = 파랑.
- `rows`는 n − 1개의 항목을 가집니다(정점 n−1은 자기 행이 없음).
- 어떤 5개 정점도 그 사이의 10개 간선이 모두 같은 색이면 안 됩니다.

검증: `python3 verify.py <witness.json>`

## 문헌 기록

R(5,5) ≥ 43: Exoo의 42-정점 채색(1989). 재구성은 *Study of Exoo's Lower
Bound for Ramsey number R(5,5)*
([arXiv:2212.12630](https://arxiv.org/abs/2212.12630))를 참고하세요. 이
witness는 아직 `records/`에 없습니다 — 유효한 42-정점 채색을 제출하면 검증
기록을 차지합니다.

## 알려진 접근법

- 간선 채색에 대한 국소 탐색(단색 K_5 개수를 에너지로 하는 담금질/타부 서치)
  으로 30대 중반까지 도달합니다.
- 최선의 구성은 대수적입니다: Paley 류 그래프와 블록 순환 채색에 국소 수리를
  결합합니다. Exoo의 42-정점 witness도 순환 구성 기반입니다.
- 빠른 이동 평가가 중요합니다: 공통 이웃 비트마스크 기법으로 각 간선을
  지나는 단색 K_5 개수를 관리하세요.
